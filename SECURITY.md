# Security Policy

## Supported Versions

This project is a learning / demonstration project.
Only the **latest version** is considered supported.

| Version | Supported |
| ------- | --------- |
| latest  | ✅ Yes    |
| older   | ❌ No     |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it using one of the following methods:

- GitHub **Security Advisory** (preferred)
- Private message to the repository owner

Please include:
- a clear description of the issue
- steps to reproduce
- potential impact

We will review the report and respond as soon as reasonably possible.

---

## Scope

This project is intended for educational and demonstration purposes.
While security best practices are followed, it is **not intended for production use** without additional hardening.
